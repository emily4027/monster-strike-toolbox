

# 🧰 怪物彈珠工具箱 (Monster Strike Toolbox)

https://emily4027.github.io/monster-strike-toolbox/

### 這是一個現代化的整合入口網頁 (Portal Page)，旨在將《怪物彈珠》的相關輔助工具（如果實分配器、抽
### 卡統計）集中管理。透過精美的互動式介面，讓使用者能快速前往各個功能子頁面。

## ✨ 主要特色

## 1. 🚀 整合入口 (Gateway)
###     集中管理：提供統一的導航介面，連結至不同的獨立工具。
###     擴充性：卡片式設計方便未來新增更多工具（如紋章計算、關卡資訊等）。
## 2. 🎨 互動式視覺體驗
###     3D 視差懸浮效果 (3D Tilt Effect)：
###        使用 JavaScript 監聽滑鼠移動事件，計算游標在卡片上的相對位置。
###        動態調整 CSS transform: rotateX/rotateY，讓卡片跟隨滑鼠角度傾斜，產生立體感。
###        裝飾圓圈具備視差位移，增強深邃感。
###     動態背景：背景採用緩慢旋轉的雷達掃描風格動畫，增加畫面的豐富度而不搶奪視覺焦點。
## 3. 📱 響應式設計 (RWD)
###     自適應佈局：使用 CSS Grid 與 Flexbox，在電腦版顯示多欄卡片，在手機版自動調整為單欄垂直排列。
###     觸控優化：按鈕與卡片尺寸皆針對手指點擊進行優化。

## 🛠 技術架構

### HTML5: 語意化標籤，結構清晰。

### CSS3:

```
CSS Variables: 統一管理主題色（綠色代表果實、藍紫色代表抽卡）。
Animations: 使用 @keyframes 製作背景與進場動畫。
Glassmorphism: 微透明的卡片背景與陰影處理。
JavaScript (Vanilla):
輕量級原生代碼，無外部依賴。
負責處理滑鼠互動邏輯與 3D 變換運算。
```
## 🚀 如何部署與設定

### 由於這是一個入口頁面，您需要將卡片的連結指向實際的工具網址。

### 1. 開啟 index.html
### 2. 修改連結 (href)： 找到以下段落，將 # 或預設文字替換為您實際部署的 GitHub Pages 網址。

## 📂 檔案結構

```
.
├── index.html # 入口首頁 (包含工具連結)
├── style.css # 樣式表 (包含 3D 特效樣式與 RWD 設定)
└── script.js # 互動腳本 (處理卡片傾斜動畫)
```
